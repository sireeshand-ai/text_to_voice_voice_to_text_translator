# Voice and Text Translator

## Overview

This project is a **Voice and Text Translator** that allows users to convert **text to speech** and **speech to text**. The application also supports **multiple language selections**, enabling users to interact with the system in different languages.

The system provides a simple interface where users can type text or speak through a microphone, and the application processes the input to generate the corresponding output.

## Features

* Convert **Text to Speech**
* Convert **Speech to Text**
* **Language selection** for different translations
* Supports **11 languages**
* Simple and easy-to-use interface

## Technologies Used

* **Python**
* **HTML**
* **CSS**

## How It Works

1. The user selects a language from the available options.
2. The user can either:

   * Enter text to convert it into speech.
   * Speak through the microphone to convert speech into text.
3. The system processes the input and generates the output in the selected language.

## Installation

1. Clone the repository.
2. Install the required libraries using the requirements file.

```
pip install -r requirements.txt
```

3. Run the application.

```
python app.py
```

4. Open the application in your browser and start using the translator.
